### Description
***[Problem Domain for the Code Challenge]***

Given an array. Return "Yes" if the sequence is considered a perfect sequence. Otherwise, return "No"

---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| [1,3,2] | Yes |
| [2,99,100] | No |
